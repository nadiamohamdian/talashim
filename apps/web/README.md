# Sadaf Gold — Storefront (Web)

فروشگاه مشتری — **پورت پیش‌فرض 3001**

```bash
pnpm --filter @sadafgold/web dev
```

باز کنید: **http://localhost:3001**

- خانه و محصولات **بدون ورود**
- ورود فقط در **تسویه حساب** (`/checkout`)

پنل ادمین: **http://localhost:3002** — [`apps/admin/README.md`](../admin/README.md)
